## Sal-Meter 오픈 공모전 개요 (Global)

Sal-Meter 오픈 공모전은  
**CAIS(Consciousness–Aptamer Interface System) 표준을 준수하는  
압타머–요오드 산화환원(redox) 센싱 기반 의식·정신 건강 측정 바이오센서**를  
개발하기 위한 **글로벌 공개 경쟁 프로젝트**입니다.

본 공모전은  
**“측정 가능한 의식 신호를 정의하고 검증할 수 있는가”**라는  
근본적 질문에 대한 **세계 최초의 개방형 실험**입니다.

- 오픈 사이언스
- 오픈 하드웨어
- 전면 공개 검증
- 독립 CRO 평가

- 팀당 최대 30만 달러
