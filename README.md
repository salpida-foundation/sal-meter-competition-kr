## 이 공모전은 무엇인가?

### 문제 정의
- 정신 건강 평가는 주관적(설문·의사 판단)에 의존
- 진단 기준 편차, 객관 데이터 부족
- AI는 의식을 이해할 **객관 지표**가 없음

### 우리의 접근
- **CAIS 과학 표준**으로 의식 상태를 측정
- 결과는 **전면 공개(CC BY-SA 4.0)**

## 3가지 참여 트랙

### 트랙 A: 전체 시스템 개발 (최고 난이도)
- **상금:** 팀당 $300,000
- **선정:** 2–3팀
- **기간:** 개발 9개월 + 검증 6개월 = 15개월
- **모집:** 2026년 4월

#### 상금 vs 연구비 (중요)
- 상금은 **완료 보상**, 전체 개발비 아님
- 예상 총비용: **~$900,000**
  - 인건비: ~$630,000 (9개월)
  - 기타: ~$270,000
- 팀은 자체 재원/기관 예산/외부 펀딩 병행 필요

### 트랙 B: 컴포넌트 개발
- **상금:** $50,000–$100,000
- **선정:** 5–10팀
- **기간:** 6–12개월
- 주제: 알고리즘, AI, 펌웨어, 앱, 클라우드, 테스트 프레임워크

### 트랙 C: 오픈 커뮤니티
- **상금:** 없음 (경험·기여 인정)
- 참여: 무제한
- 기여: 프로토타입, 문서·번역, 오픈소스, 논문, 하드웨어, 규제 가이드
