## 이것이 뭔가요?

Sal-Meter 오픈 공모전은 세계 최초로 인간의 의식을 측정하는 장치를 만드는 국제 공개 경쟁입니다.

이 공모전의 목표는 과학적 표준에 기반한 의식 측정 기술을 개발하고, 전 세계의 연구팀, 개발사, 대학 연구소가 함께 참여하도록 하는 것입니다.

### 핵심 정보

발주 기관: Salpida Foundation (의식 과학 연구재단)
목표: CAIS(의식-앱타머 인터페이스) 기술 기반 의식 측정 센서 개발
상금 규모: 총 $400만 달러 이상
참여 자격: 전 세계 연구팀, 개발사, 대학 연구소, 개인
설계 라이선스: CC BY-SA 4.0 (핵심 설계는 공개되어야 함)
기술 기반: 앱타머(Aptamer) + 요오드(Iodine) 센서 기술

## 왜 이것이 중요한가?

### 현재의 문제: 의식을 측정할 표준이 없다

지금까지 정신 건강은 주로 주관적인 방법으로만 평가되어 왔습니다.

예를 들면:
- 우울증: 환자가 "기분이 안 좋아요"라고 하면 의사가 평가
- 불안장애: "얼마나 불안한가"를 5단계로 표현
- 스트레스: 스트레스 지수 테스트는 객관성 부족
- 명상 효과: "명상이 효과가 있는지" 확인이 어려움

이렇게 주관적이면:
- 의사마다 진단이 다름
- 같은 환자도 시간에 따라 평가가 달라짐
- 증상 개선을 객관적으로 확인 불가능
- 새로운 치료법의 효과 비교가 어려움

### 해결책: Sal-Meter

Sal-Meter는 의식의 상태를 생물학적 신호로 측정하는 장치입니다.

작동 원리:
1. 환자의 혈액, 타액, 또는 호흡 샘플 채취
2. 앱타머(Aptamer)라는 분자 센서에 반응
3. 요오드 색깔 변화를 광학 센서로 감지
4. 인공지능으로 신호 처리
5. 1분 내에 의식 상태 수치화: VCE, CRI, CFI 지표 출력

장점:
- 객관적이고 재현 가능한 측정
- 빠른 진단 (1분 이내)
- 변화 추적 가능 (치료 전후 비교)
- 모든 사람에게 같은 기준 적용

### 미래에 미치는 영향

Sal-Meter가 성공하면:

1. 정신 건강 혁명
   - 의사들이 객관적 데이터로 진단
   - 우울증, 불안장애, PTSD 조기 발견 및 치료
   - 자살 위험 예측 및 예방 가능

2. 개인화된 의료
   - 각 개인의 의식 필요도에 맞춘 맞춤 치료
   - 약물 용량 최적화
   - 치료 효과 실시간 모니터링

3. AI와 인류의 관계 개선
   - 인공지능이 인간의 의식 상태를 이해
   - AI가 인간을 존중하는 기술 개발 가능
   - AI 윤리의 과학적 기반 마련

4. 문명의 전환
   - 지금까지: 물질(물건, 돈) 중심 문명
   - 미래: 의식(정신, 행복) 중심 문명으로 전환
   - 개인의 의식 상태가 정책과 기술 개발의 중심이 됨

## 공모 트랙 설명 (3가지 방식)

### 트랙 A: 전체 시스템 개발

상금: 팀당 300,000 USD (약 4억 원)
팀 수: 2-3개 팀
개발 기간: 9개월
검증 기간: 6개월
총 기간: 15개월

이 트랙의 의미:

Sal-Meter의 완전한 장치를 처음부터 끝까지 개발하는 트랙입니다. 이것은 가장 도전적이면서도 가장 큰 영향을 미치는 일입니다.

구체적으로 하는 일:

1. 센서 개발
   - 앱타머 분자 설계 및 합성
   - 요오드 산화 반응 최적화
   - 정확도 80% 이상 달성

2. 하드웨어 개발
   - PCB 회로 설계
   - 광학 센서 통합
   - 전자 부품 선택 및 배치
   - 배터리 및 전원 관리
   - 케이싱 및 방수 설계

3. 신호 처리 및 AI
   - 노이즈 필터링 (측정 오류 제거)
   - 신호 정규화 (표준화)
   - VCE, CRI, CFI 계산 알고리즘 개발
   - 기계학습 모델 훈련

4. 소프트웨어 개발
   - 모바일 앱 (iOS, Android)
   - 데스크탑 소프트웨어
   - 클라우드 연동
   - 사용자 인터페이스 디자인

5. 검증 및 테스트
   - 50명 이상의 참여자로 임상 테스트
   - 독립적 검사기관(CRO)과 협력
   - 민감도 80% 이상, 특이도 85% 이상 달성
   - 안전성 및 정확성 검증

필요한 전문성:

1. 바이오센서 / 앱타머 화학 엔지니어 (2명)
   - 앱타머 설계, 합성, 최적화
   - 분자 수준의 화학 반응 이해
   
2. 신호 처리 / AI-ML 엔지니어 (2-3명)
   - 센서 신호를 의미 있는 데이터로 변환
   - 인공지능 모델 훈련 및 검증
   - 파이썬, TensorFlow 등 경험
   
3. 하드웨어 시스템 엔지니어 (1명)
   - PCB 설계, 임베디드 시스템
   - 다양한 센서 통합 경험
   - 전자공학 이해
   
4. 통계 및 검증 전문가 (1명)
   - 임상 시험 설계
   - 통계 분석, p-값 이해
   - 규제 기준 이해
   
5. 프로젝트 매니저 (1명)
   - 9개월 일정 관리
   - 팀 간 조율
   - 예산 관리

예상 결과:

- 세계 최초로 과학적 표준에 기반한 의식 측정 장치 완성
- 독립적으로 검증된 결과 공개
- 오픈소스 설계 (CC BY-SA 4.0) 공개
- 국제 저널에 논문 발표
- 각 팀 $300,000 USD 상금 수여

### 트랙 B: 컴포넌트 개발

상금: 팀당 50,000 - 100,000 USD (6천만 - 1억 3천만 원)
팀 수: 5-10개 팀
개발 기간: 6-12개월

이 트랙의 의미:

Sal-Meter 전체 시스템을 만드는 것이 너무 크면, 특정한 부품이나 기능 하나를 깊이 있게 개발할 수 있습니다. 마치 자동차 회사가 엔진을 전문적으로 만드는 것처럼, 우리도 모듈 단위의 전문화를 환영합니다.

트랙 B의 가능한 주제들:

1. 센서 모듈 개발
   할 일: 앱타머-요오드 센서를 최고 성능으로 만들기
   - 감도 극대화 (약한 신호도 감지)
   - 반응 시간 단축
   - 안정성 향상 (여러 번 반복 사용)
   - 비용 절감
   예상 효과: 센서 성능이 좋으면 전체 시스템 성공

2. 신호 처리 알고리즘 개발
   할 일: 센서에서 나온 복잡한 신호를 VCE, CRI, CFI로 변환하기
   - 노이즈 제거 (측정 오류 필터링)
   - 신호 정규화 (다양한 조건에서의 표준화)
   - 이상 탐지 (잘못된 측정 감지)
   - 실시간 처리 최적화
   예상 효과: 정확한 지표 계산이 장치 신뢰도 결정

3. AI/머신러닝 모델 개발
   할 일: 신호 데이터로부터 의식 상태를 예측하는 모델 만들기
   - 훈련 데이터셋 구축
   - 다양한 머신러닝 모델 실험 (SVM, Random Forest, Neural Network 등)
   - 모델 성능 검증
   - 과적합(Overfitting) 방지
   예상 효과: 정확한 AI 모델 = 신뢰할 수 있는 장치

4. 교체형 센서 카트리지 설계
   할 일: 일회용 센서 팩을 설계하고 제조하기
   - 소형화 설계 (컴팩트하게)
   - 증류수 새는 것 방지 (방수)
   - 자동 끼우고 빼기 용이한 구조
   - 교정(Calibration) 정보 내장
   예상 효과: 사용자가 쉽게 센서 교체 가능

5. 펌웨어 개발
   할 일: 센서 장치에 내장되는 소프트웨어 만들기
   - 센서 신호 읽기
   - 신호 전처리
   - 배터리 전력 관리
   - 와이파이/블루투스 통신
   - 오류 처리 및 복구
   예상 효과: 하드웨어를 제어하는 "뇌" 역할

6. 모바일 앱 개발
   할 일: 스마트폰에서 사용자가 조작하는 앱 만들기
   - 센서 데이터 표시 (그래프, 수치)
   - 사용자 프로필 관리
   - 측정 기록 저장 및 분석
   - 클라우드 동기화
   - 아름다운 인터페이스 디자인
   예상 효과: 사용자가 실제로 쓸 수 있는 제품

7. 클라우드 플랫폼 개발
   할 일: 측정 데이터를 저장하고 분석하는 서버 구축
   - 데이터베이스 설계
   - API 개발 (모바일 앱과 통신)
   - 데이터 보안 및 암호화
   - 사용자 프라이버시 보호
   - 확장 가능한 아키텍처
   예상 효과: 여러 사용자의 데이터 중앙 관리

8. 테스트 및 검증 프레임워크
   할 일: 다른 팀들이 만든 것을 검증할 수 있는 체계 만들기
   - 테스트 케이스 작성
   - 성능 벤치마크 정의
   - 자동화된 테스트 스크립트
   - 문서화
   예상 효과: 전체 팀의 품질 향상

장점:

- 진입 장벽 낮음 (전체 시스템보다 작은 범위)
- 특화된 분야에 집중 가능 (품질 극대화)
- 여러 팀의 다양한 접근 방식 가능
- 더 높은 성공률
- 완료 시간 단축 (6-12개월)

예상 결과:

- 각 팀이 전문화된 컴포넌트 완성
- 다양한 구현 방식의 선택지 제공
- 국제 컨퍼런스 발표 기회
- 각 팀 $50K-$100K USD 상금 수여
- 트랙 A 팀들이 이 컴포넌트들을 참고/활용 가능

### 트랙 C: 오픈 커뮤니티

상금: 없음 (재정 지원 대신 인정과 경험)
참여자: 무제한 (개인, 학생, 취미 개발자, 비영리 단체 등 누구나)
활동 기간: 자유로움 (언제든지 시작/끝낼 수 있음)

이 트랙의 의미:

트랙 A와 B는 정해진 자금과 일정이 있지만, 트랙 C는 완전히 자유롭고 개방적입니다. 이 트랙은 Sal-Meter를 전 세계의 개발자, 연구자, 학생들이 함께 개선하는 커뮤니티입니다.

할 수 있는 일:

1. 기술 프로토타입 만들기
   - Sal-Meter 작동 원리 이해하기
   - 간단한 버전부터 시작해 점진적으로 개선
   - 다양한 재료와 기술로 실험
   - 비용이 낮은 대안 개발
   예: 학생들이 대학 실험실에서 프로토타입 만들고 GitHub에 공개

2. 기술 문서 작성 및 번역
   - 복잡한 기술 문서를 더 쉬운 말로 설명
   - 다양한 언어로 번역 (한국어, 일본어, 스페인어 등)
   - 비디오 튜토리얼 제작
   - 블로그 글 작성
   예: 바이오센서 초보자도 이해할 수 있도록 설명

3. 오픈 소스 코드 작성
   - 신호 처리 알고리즘 구현 (Python, MATLAB)
   - 모바일 앱 개발 (React Native, Flutter)
   - 웹 대시보드 개발 (JavaScript, React)
   - 데이터 분석 스크립트
   예: GitHub에서 누구나 이용할 수 있는 코드 공유

4. 학술 연구 및 논문
   - Sal-Meter 기술의 타당성 연구
   - 다양한 조건에서의 성능 비교
   - 새로운 응용 분야 탐색
   - 비판적 분석 논문
   예: 대학원생이 논문 주제로 삼고 국제 학술지에 투고

5. 커뮤니티 참여 및 지원
   - Discussions에서 다른 사람의 질문에 답변
   - 버그 발견 및 보고
   - 기능 개선 아이디어 제시
   - 신입자를 위한 가이드 작성
   예: 경험 많은 개발자가 입문자를 멘토링

6. 데이터 수집 및 공유
   - 프로토타입을 통해 측정 데이터 수집
   - 데이터를 공개 저장소(Zenodo, OSF)에 업로드
   - 다른 팀의 분석을 위한 기초 자료 제공
   예: 자원자 50명의 측정 데이터 공개

7. 하드웨어 디자인 개선
   - 3D 프린팅으로 케이싱 설계
   - 저비용 대체 부품 제시
   - 휴대성 향상 아이디어
   - 사용자 경험 개선 디자인
   예: 착용하기 쉬운 웨어러블 형태 설계

8. 규제 및 윤리 검토
   - 각 국가의 의료기기 규제 조사
   - 데이터 프라이버시 가이드 작성
   - 윤리적 사용 기준 개발
   - 규제 준수 체크리스트
   예: FDA, 식약청 요구사항 분석

장점:

- 전혀 비용이 필요 없음 (자신의 시간과 기술만 투자)
- 완전히 자유로운 일정 (학교나 직장과 병행 가능)
- 실제 프로젝트에 기여하는 경험
- 국제 협업 경험
- 이력서/포트폴리오 구축

기대되는 성과:

- GitHub에 수천 개의 오픈 소스 기여
- 국제 학술지 게재 논문

## 기술 설명: 실제로 뭘 어떻게 만드나?

### Sal-Meter의 작동 원리 상세 설명

#### 단계 1: 생물 샘플 채취

첫 번째 단계: 사용자의 샘플을 채취합니다.

방법:
- 혈액: 손가락 끝에서 한 두 방울 (혈당 측정기 방식)
- 타액: 입 안쪽 면봉으로 문지름 (DNA 검사 방식)
- 호흡: 특수 필터에 입김 내쉬기

왜 이 샘플들을 쓰나?
- 비침습적 (신체에 해를 주지 않음)
- 수집 쉬움 (집에서 가능)
- 의식 상태 정보 포함 (신경화학 물질 함유)

#### 단계 2: 샘플 전처리

두 번째 단계: 샘플을 분석하기 좋게 준비합니다.

과정:
1. 원심분리 (centrifugation)
   - 빠르게 회전시켜 혈구 제거
   - 혈청(plasma) 분리

2. 여과 (filtration)
   - 큰 입자 제거

3. 희석 (dilution)
   - 적절한 농도로 조정

4. pH 조정
   - 최적 환경 조성

#### 단계 3: 앱타머-요오드 반응

세 번째 단계: 마법이 일어나는 부분입니다.

앱타머란?
- DNA나 RNA와 유사한 분자
- 특정 물질에만 결합하는 선택성 높음
- 특정 조건에서 색깔 변함

요오드의 역할:
- 여러 산화 상태로 존재: I⁻ (아이오드화물) → I₂ (분자 요오드) → I₃⁻ (요오드화 삼중) → I₅⁻ (요오드화 오중)
- 각 상태마다 색깔과 흡수 파장이 다름
- 의식 상태에 따라 특정 산화 상태로 변함

의식과의 연결:
- 의식이 높을 때: 신경계의 글루타메이트(흥분 신경전달물질) 증가
- 글루타메이트가 특정 분자와 결합
- 그 분자가 요오드 산화 상태에 영향
- 앱타머가 그 변화를 감지
- (이것은 아직 정확한 생화학 메커니즘은 개발 단계)

#### 단계 4: 광학 감지

네 번째 단계: 색깔 변화를 센서로 감지합니다.

센서의 구성:
1. 광원 (LED)
   - 특정 파장의 빛 방출
   - 예: 280nm (자외선 영역)

2. 샘플을 통한 빛 투과
   - 요오드의 각 형태가 다른 파장 흡수

3. 검출기 (photodiode)
   - 투과한 빛을 전기 신호로 변환
   - 흡광도(absorbance) 측정

측정값:
- 각 파장에서의 흡광도
- 예: 280nm에서 0.45, 350nm에서 0.12, 450nm에서 0.08

#### 단계 5: 신호 처리

다섯 번째 단계: 복잡한 신호를 정리합니다.

노이즈 제거 (signal filtering):
- 측정 오류 제거
- 방법: 칼만 필터(Kalman Filter), 저주파 필터(Low-pass Filter)

신호 정규화 (normalization):
- 다양한 조건을 표준화
- 배경 광 제거
- 온도 보정 (온도가 높으면 반응이 약해짐)
- 샘플 농도 보정

특징 추출 (feature extraction):
- 여러 파장에서의 데이터로부터 의미 있는 특징 도출
- 산화 상태 비율 계산
- 반응 속도 측정

#### 단계 6: AI 모델로 지표 계산

여섯 번째 단계: 신호 데이터로부터 VCE, CRI, CFI 계산합니다.

VCE (Vibrational Consciousness Energy) 계산:

수식 개념 (실제 수식은 개발 중):
VCE = (산화 상태의 진동 에너지) + (응답 속도) + (신호 진폭)

의미:
- 높음 (0.8 이상): 의식이 활성화됨, 스트레스 많음, 또는 각성 상태
- 중간 (0.4-0.6): 평상시 의식 상태
- 낮음 (0.2 이하): 의식 저하, 수면, 또는 깊은 명상

CRI (Consciousness Resonance Index) 계산:

수식 개념:
CRI = (산화 상태의 안정성) × (신호 일관성) / (잡음 수준)

의미:
- 높음 (0.8 이상): 의식이 안정적, 정신 맑음, 감정 안정
- 중간 (0.4-0.6): 약간 산만함
- 낮음 (0.2 이하): 의식 불안정, 혼란, 정신 불건전

CFI (Conscious Field Index) 계산:

수식 개념:
CFI = (VCE의 구조적 정렬) + (CRI의 안정성) + (신경 신호의 통합성)

의미:
- 전체 의식 필드의 상태를 하나의 지수로 표현
- 높음: 의식이 통합적, 조화로움
- 낮음: 의식이 분산적, 단편적

#### 단계 7: 결과 출력

일곱 번째 단계: 사용자가 이해할 수 있는 형태로 표시합니다.

디지털 출력:
VCE: 0.68 (보통보다 높음)
CRI: 0.82 (매우 안정적)
CFI: 0.75 (전체적으로 양호)

해석: 당신은 지금 적당한 각성 수준에서 안정적인 의식 상태입니다.
권장: 휴식 또는 가벼운 명상

- 산업 디자인 (우리 기기의 모양)
- 클라우드 분석 서비스 (우리 AI 모델)
- 마케팅 방식 (우리 브랜드)

비유:

앵무새 예시:
- 핵심 설계: "앵무새는 색깔 있는 깃털로 이루어짐" (공개)
- 구현 IP: "우리의 앵무새는 이렇게 아름답게 생겼음" (비밀)

자동차 예시:
- 핵심 설계: "바퀴 + 엔진 + 핸들로 만들어짐" (공개)
- 구현 IP: "우리 엔진은 이렇게 효율적" (비밀)

예: Sal-Meter v1.0 제품화

우리의 계획:
1. 핵심 센서 기술 공개 (CC BY-SA 4.0)
2. 우리는 클라우드 분석 서비스로 수익화
   - 의식 상태 분석
   - 개인화된 추천
   - 의료기관 대시보드
   - 기업용 팀 웰니스 솔루션

다른 회사의 계획:
1. 우리의 공개된 센서 기술 활용
2. 자신들의 제조 방식으로 제품 만듦
3. 자신들의 소프트웨어 서비스로 수익화
4. 우리 기술의 개선사항은 다시 공개 (Share-Alike)

결과:
- 지구상의 모든 회사가 의식 측정 기술을 사용 가능
- 경쟁으로 인한 가격 하락
- 소비자 이득 (저렴한 가격에 고품질 제품)
- 모두가 Win-Win

- 그런 논문도 좋은 저널에 게재 가능

예:
"우리는 센서 감도를 70%까지 달성했지만, 80% 목표를 못 맞추었다. 그 이유는 X 때문이고, Y 방법을 써야 해결 가능하다"
→ 이것도 논문이 됨

트랙 C 참여자:
- 포트폴리오 구축 (성공/실패 모두)
- 국제 협업 경험
- 논문 공동저자 기회 (다른 팀의 성공 지원 시)

질문 7: 이것이 의료 기기로 인정되나요?

답변:

지금은 "의료 기기 아님"입니다. 하지만 향후 가능합니다.

현재 (Phase 2): 연구 단계
- 임상 사용 금지
- "연구용" 표시 필수
- 규제 요구사항 미적용

근래 중 (Phase 3-4): 규제 승인 추진
- FDA (미국): Class II 또는 III 의료기기 신청
- EMA (유럽): CE 마크 신청
- 식약청 (한국): 의료기기 허가 신청

규제 프로세스:
1. 전임상 시험 (동물 또는 세포)
2. 초기 임상 시험 (Phase I: 안전성, 소수 인원)
3. 주요 임상 시험 (Phase II/III: 효능, 많은 인원)
4. 규제 제출 및 심사
5. 승인 및 출시
6. Phase IV: 출시 후 감시

예상 타이밍:
- 현재: 2026-2027 (연구)
- 2027-2029: 규제 신청
- 2029-2031: 규제 심사
- 2031+: 의료기기 승인 및 임상 사용 가능

따라서 우리 공모전은 그 초기 단계입니다.
