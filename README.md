## 이 공모전은 무엇인가?

Sal-Meter 오픈 공모전은 공개 표준(DOI) 기반 의식 측정 장치 개발 국제 경쟁입니다.

### 문제 정의

현재 정신 건강은 주로 주관적 평가(설문, 의사 판단)에 의존합니다.

의사마다 진단 기준이 다름
객관적 데이터가 부족해 치료 효과 추적 곤란
AI가 인간 의식을 이해할 수 없음

### 우리의 접근

과학적 표준(CAIS 기술)으로 의식 상태를 객관적으로 측정하고, 모든 결과를 공개(CC BY-SA 4.0)합니다.

### 핵심 기술

| 항목 | 설명 |
|------|------|
| 센서 | 앱타머(Aptamer) + 요오드(Iodine) 산화 상태 감지 |
| 측정 | 1분 내 추정 결과 출력 (VCE, CRI, CFI 지표) |
| 검증 | 독립 검사기관(CRO) 검증 예정 |
| 라이선스 | CC BY-SA 4.0 (핵심 설계는 공개 필수) |

## 3가지 참여 방식 (트랙)

### 트랙 A: 전체 시스템 개발 (가장 도전적)

상금: 팀당 300,000 USD (약 3억 9,000만 원)
팀 수: 2-3개 팀
기간: 9개월 개발 + 6개월 검증 = 15개월
모집 시작: 2026년 4월

상금 vs 연구 지원금 (매우 중요)

주의: 명확한 구분

상금($300K): 개발 완료 시 지급되는 보상
실제 개발 비용: 팀이 별도로 조달해야 함

예상 인건비: 630,000 USD (9개월)
예상 기타 비용: 270,000 USD (부품, 실험실, 출장)
총 필요 자금: 약 900,000 USD

이것은 무엇을 의미하나?

SICS의 300,000 USD는 상금이지, 팀의 전체 개발비가 아닙니다.
팀은 자체 기관 지원, 펀딩, 또는 자체 재원을 병행해야 합니다.

예시: 대학 연구실이면 학교 예산 사용, 회사면 회사 R&D 예산 사용

필요한 팀 구성

7명 + 자문진:
바이오센서/화학 박사 2명
신호처리/AI 엔지니어 2-3명
하드웨어 엔지니어 1명
통계/검증 전문가 1명
프로젝트 매니저 1명

월급 기준 (팀이 자체 책정)

이것은 참고용 미국 임금입니다. 각 팀이 자체 예산에 맞춰 책정합니다.

| 직급 | 월급 USD | 월급 한화 | 9개월 총액 USD | 9개월 총액 한화 |
|------|---------|---------|---------------|---------------|
| 박사/박사후 | 8,000-12,000 | 약 1,040만-1,560만 원 | 72,000-108,000 | 약 9,360만-1억 4,040만 원 |
| 엔지니어 | 6,000-10,000 | 약 780만-1,300만 원 | 54,000-90,000 | 약 7,020만-1억 1,700만 원 |
| PM | 7,000-9,000 | 약 910만-1,170만 원 | 63,000-81,000 | 약 8,190만-1억 530만 원 |

### 트랙 B: 컴포넌트 개발 (특화 분야)

상금: 팀당 50,000-100,000 USD (약 6,500만-1억 3,000만 원)
팀 수: 5-10개 팀
기간: 6-12개월
모집 시작: 2026년 4월

가능 주제:
신호 처리 알고리즘 개발
AI/머신러닝 모델 개발
펌웨어 개발
모바일 앱 개발
클라우드 플랫폼 개발
테스트 프레임워크 구축

---

### 트랙 C: 오픈 커뮤니티 (자유로운 참여)

상금: 없음 (대신 인정과 경험)
참여자: 무제한 (개인, 학생, 취미 개발자 등)
기간: 자유 (언제든 시작/종료)

할 수 있는 일:
프로토타입 제작
기술 문서 작성 및 번역
오픈소스 코드 개발
학술 논문
하드웨어 디자인
규제 가이드 작성

## 자주 묻는 질문 (FAQ)

### Q1. 경험이 없어도 지원 가능한가?

A. 트랙 C: 네, 가능합니다.
트랙 A/B: 관련 분야 경험 권장 (바이오센서 3+ 년, AI 2+ 년 등)

### Q2. 한국 팀도 지원 가능한가?

A. 네, 절대적으로 가능합니다. 국가 제한 없습니다.

### Q3. 우리 팀이 만든 기술을 팔 수 있나?

A. 네, 하지만 조건이 있습니다:

공개 필수 (CC BY-SA 4.0):
앓타머 센서 기술
신호 처리 알고리즘
VCE/CRI/CFI 계산 방식

독점 가능:
제조 공정
클라우드 분석 서비스
산업 디자인
마케팅 방식

### Q4. 상금으로 개발비를 전부 충당할 수 있나?

A. 아니요. 상금 300,000 USD는 보상이지, 전체 개발비가 아닙니다.

팀의 실제 필요 자금(약 900,000 USD)은 다음으로 충당해야 합니다:
대학 연구비
회사 R&D 예산
외부 펀딩
자체 재원
