# 기술 상세 설명: Sal-Meter 작동 원리

## 개요

Sal-Meter는 앱타머(Aptamer) 기반 센서와 요오드(Iodine) 산화 상태 감지를 통해 의식 상태를 측정하는 장치입니다.

이 문서는 Sal-Meter의 작동 방식을 7단계로 설명합니다.

---

## 1단계: 생물 샘플 채취

첫 번째 단계: 사용자의 샘플을 채취합니다.

### 샘플 수집 방법

- **혈액**: 손가락 끝에서 한두 방울 (혈당 측정기 방식)
- **타액**: 입 안쪽 면봉으로 문지름 (DNA 검사 방식)
- **호흡**: 특수 필터에 입김 내쉬기

### 왜 이 샘플들인가?

- 비침습적 (신체에 해를 주지 않음)
- 수집이 쉬움 (집에서도 가능)
- 의식 상태 정보 포함 (신경화학 물질 함유)

---

## 2단계: 샘플 전처리

샘플을 분석하기 좋게 준비합니다.

### 전처리 과정

1. **원심분리 (Centrifugation)**
   - 빠르게 회전시켜 혈구 제거
   - 혈청(Plasma) 분리

2. **여과 (Filtration)**
   - 큰 입자 제거
   - 0.2 마이크론 필터 사용

3. **희석 (Dilution)**
   - 적절한 농도로 조정
   - 보통 1:10 희석

4. **pH 조정**
   - 최적 환경 조성 (pH 7.4)
   - 인산완충액(PBS) 사용

---

## 3단계: 앱타머-요오드 반응

**마법이 일어나는 부분입니다.**

### 앱타머(Aptamer)란?

- DNA나 RNA와 유사한 분자
- 특정 물질에만 결합하는 높은 선택성
- 특정 조건에서 색깔 변함

### 요오드의 역할

요오드는 여러 산화 상태로 존재합니다:

- **I⁻** (아이오드화물) → 무색
- **I₂** (분자 요오드) → 갈색
- **I₃⁻** (요오드화 삼중이온) → 갈색
- **I₅⁻** (요오드화 오중이온) → 진한 갈색

각 상태마다 **색깔과 흡수 파장이 다릅니다.**

### 의식 상태와의 연결

- 의식이 높을 때: 신경계의 글루타메이트(흥분 신경전달물질) 증가
- 글루타메이트가 특정 분자와 결합
- 그 분자가 요오드 산화 상태에 영향
- **앱타머가 그 변화를 감지**

*(주: 정확한 생화학 메커니즘은 개발 중입니다)*

---

## 4단계: 광학 감지

색깔 변화를 센서로 감지합니다.

### 광학 센서 구성

1. **광원 (LED)**
   - 특정 파장의 빛 방출
   - 예: 280nm (자외선 영역), 450nm (가시광선)

2. **샘플을 통한 빛 투과**
   - 요오드의 각 형태가 다른 파장 흡수
   - 일부는 투과, 일부는 흡수됨

3. **검출기 (Photodiode)**
   - 투과한 빛을 전기 신호로 변환
   - 흡광도(Absorbance) 측정

### 측정 데이터 예시

280nm 파장: 흡광도 0.45
350nm 파장: 흡광도 0.12
450nm 파장: 흡광도 0.08


---

## 5단계: 신호 처리

복잡한 신호를 정리합니다.

### 노이즈 제거 (Signal Filtering)

- 측정 오류 제거
- 칼만 필터(Kalman Filter) 사용
- 저주파 필터(Low-pass Filter) 적용

### 신호 정규화 (Normalization)

다양한 조건을 표준화합니다:

- 배경 광 제거
- 온도 보정 (온도가 높으면 반응이 약해짐)
- 샘플 농도 보정

### 특징 추출 (Feature Extraction)

여러 파장 데이터로부터 의미 있는 특징 도출:

- 산화 상태 비율 계산
- 반응 속도 측정
- 신호 진폭 계산

---

## 6단계: AI 모델로 지표 계산

신호 데이터로부터 3가지 지표를 계산합니다.

### VCE (Vibrational Consciousness Energy)

**의식의 진동 에너지 지표**

계산 방식 (개념):

VCE = (산화 상태의 진동 에너지)
+ (응답 속도)
+ (신호 진폭)


**해석:**
- 0.8 이상: 의식 활성화, 스트레스 많음, 각성 상태
- 0.4-0.6: 평상시 의식 상태
- 0.2 이하: 의식 저하, 수면, 깊은 명상

### CRI (Consciousness Resonance Index)

**의식 공명 지수 - 안정성 측정**

계산 방식 (개념):
CRI = (산화 상태의 안정성)
× (신호 일관성)
/ (잡음 수준)

**해석:**
- 0.8 이상: 의식 안정, 정신 맑음, 감정 안정
- 0.4-0.6: 약간 산만함
- 0.2 이하: 의식 불안정, 혼란, 정신 불건전

### CFI (Conscious Field Index)

**의식 필드 지수 - 통합성 측정**

계산 방식 (개념):
CFI = (VCE의 구조적 정렬)
+ (CRI의 안정성)
+ (신경 신호의 통합성)


**해석:**
- 높음: 의식이 통합적, 조화로움
- 낮음: 의식이 분산적, 단편적

### 모델 학습

머신러닝 모델은 다음을 통해 훈련됩니다:

1. 많은 샘플의 센서 신호 데이터
2. 각 샘플의 알려진 의식 상태
3. 신경망이 신호 → 지표 매핑 학습
4. 새로운 신호에 대해 지표 예측
