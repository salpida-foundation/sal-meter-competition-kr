# 개발 일정: 4 Phase 상세 계획

## 개요

Sal-Meter 공모전은 4개의 단계(Phase)로 진행됩니다.

---

## Phase 0: 준비 및 문서 확정 (2026년 1-3월)

**목표**: 모든 규칙과 기술 표준 확정

### 할 일

1. 4개 필수 문서 최종 검토 및 승인
2. 기술 사양서 완성
3. 공모 시스템 구축 (지원 웹사이트, 평가 기준 확정)
4. 홍보 및 안내
5. 국제 협력 기관과 협력 체계 구축

### 예상 결과

- 모든 문서 DOI 등록 완료
- 공모 시스템 준비 완료
- 심사위원회 구성 완료

---

## Phase 1: 팀 모집 및 선정 (2026년 4-6월)

**목표**: 최우수 팀들 확보

### 조직 역할

**Track A/B 모집:**
- 지원서 모집 시작 (3개월 진행)
- 지원팀 검토 (기술력, 경험, 팀 구성)
- 상위 팀 인터뷰
- 최종 선정 (Track A 2-3팀, Track B 5-10팀)
- 선정 팀에 상세 기술 브리핑

**Track C 시작:**
- Track C 참여자들의 초기 프로토타입 공개 시작
- GitHub Discussions 활성화
- 주간 공개 세미나 시작

### 예상 결과

- Track A: 3개 팀 최종 선정 ($900K 배분)
- Track B: 7개 팀 최종 선정 ($500K 배분)
- Track C: 20+ 개 팀/개인 활동 시작

---

## Phase 2: 개발 단계 (2026년 7월 - 2027년 3월)

**기간**: 9개월
**목표**: 작동 가능한 프로토타입 완성

### Track A 팀 진행 상황

#### 월 1-2 (7-8월): 초기 설계

- 센서 재료 선정
- PCB 레이아웃 초안
- 소프트웨어 아키텍처 설계
- 프로토타입 부품 조달

**진행 상황 보고: 월 1회**

#### 월 3-5 (9-11월): 센서 개발

- 앱타머-요오드 반응 최적화
- 신호 처리 알고리즘 개발
- 초기 테스트 (자신들이 시험 대상)

**진행 상황 보고: 월 2회**

#### 월 6-7 (12월-1월): 통합 및 테스트

- 센서 + 하드웨어 + 소프트웨어 통합
- 자체 검증 (팀 20명 대상)
- 버그 수정

**진행 상황 보고: 주 1회**

#### 월 8-9 (2-3월): 성능 최적화

- 정확도 개선 (목표: 민감도 80%, 특이도 85%)
- 신뢰도 개선
- 최종 조정

**진행 상황 보고: 주 1회**

### Track B 팀 진행 상황

| 월 | 활동 |
|----|------|
| 1-3 | 컴포넌트 정의 및 기초 개발 |
| 4-6 | 핵심 기능 구현 |
| 7-9 | 성능 최적화 및 문서화 |
| 10-12 | 최종 검증 |

### Track C 커뮤니티

- 매주 기술 세미나 개최
- GitHub에 지속적인 코드 업로드
- 블로그 글, 비디오, 튜토리얼 공개
- 국제 학술지에 논문 투고 시작

## Phase 3: 검증 단계 (2027년 4-9월)

**기간**: 6개월
**목표**: 독립적 검증 완료

### CRO (Contract Research Organization) 역할

#### 1단계: 참여자 모집 (총 50명)

- 25명: 높은 VCE 예상 (스트레스 많은 상태)
- 25명: 낮은 VCE 예상 (이완 상태)

#### 2단계: 사전 평가

- 기본 건강 검진
- 정신 건강 설문
- 기준선 측정

#### 3단계: 측정 프로토콜

- 각 참여자 최소 3회 측정 (다양한 상태에서)
- 동시에 표준 도구로도 평가 (비교용)
- 블라인드 평가 (측정자가 참여자 상태 모름)

#### 4단계: 데이터 분석

**민감도 검증:**
- 실제 스트레스 있는 25명 중 20명 이상 감지?
- 목표: 80% 이상

**특이도 검증:**
- 스트레스 없는 25명 중 21명 이상을 정확히 "없음"으로 판단?
- 목표: 85% 이상

**기타 통계 검증:**
- 신뢰도 구간
- p-값 계산
- 재현성 확인

#### 5단계: 보고서 작성

- 방법론 공개
- 모든 데이터 공개
- 부정적 결과도 포함

### 결과 발표

- GitHub에 모든 데이터 업로드
- OSF(Open Science Framework)에 논문 등록
- 독립 저널에 투고

## Phase 4: 규제 경로 및 다중 사이트 검증 (2027년 10월+)

**목표**: 국제적 인정 및 규제 승인 추진

### 할 일

#### 1. 규제 기관 상담

- **FDA (미국)**: Class II 또는 III 의료기기 경로
- **EMA (유럽)**: CE 마크 신청
- **식약청 (한국)**: 의료기기 허가 신청
- **PMDA (일본)**: 일본 규제 경로

#### 2. 다중 사이트 검증

- 3-5개 독립 기관에서 동시 검증
- 다양한 인종, 나이, 성별로 확대
- 다양한 지역에서 데이터 수집

#### 3. 임상 시험 계획

- **Phase I**: 안전성 테스트 (20-50명, 3-6개월)
- **Phase II**: 효능 테스트 (100-300명, 6-12개월)
- **Phase III**: 대규모 효능 검증 (1,000명+, 1-2년)
- **Phase IV**: 출시 후 감시 (지속)

#### 4. 규제 제출

- Phase I/II/III 데이터 제출
- 안전성 보고서 제출
- 기술 문서 제출
- 제조 기준 문서 (CMC) 제출

### 예상 결과

- 각 지역별 규제 승인 (2027-2031)
- 의료기기로 공식 인증
- 임상 환경에서의 실제 사용 가능
